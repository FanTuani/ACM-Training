# 异或运算的性质

1. 交换律

2. 结合律（即`(a^b)^c == a^(b^c)`）

3. `x^x=0`，`x^0=x`

4. **自反性 `A xor B xor B = A xor 0 = A`**

# SG 定理

## SG 函数

SG(i) = i 的全部后继节点 SG 值的 mex。其中，必败节点的 SG 值为 0。
SG(i) == 0 必败，SG(i) != 0 必胜。
因为若 SG(i) != 0，则说明必然存在 i 的后继节点 j 满足 SG(j) == 0，即必败。 

## SG 定理

若一个 ICG（公平组合游戏）由若干独立的 ICG 子游戏组成，则有：
SG(总) == SG(分1) ^ SG(分2) ^ SG(分3) ^ ...
例：Nim 博弈
